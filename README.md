
# Electric Vehicle Sales Analysis (India)

## Project Overview

This project analyzes Electric Vehicle (EV) sales data across various Indian states to understand adoption patterns and sales trends. The analysis focuses on identifying regional variations, growth behavior, and overall market insights using data analytics and basic machine learning techniques.

---

## Dataset Description

The dataset contains state-wise EV sales records along with time-related attributes. Key preprocessing steps included data cleaning, handling missing values, and preparing features for analysis.

---

## Objectives

* Study EV sales distribution across states
* Identify sales trends and patterns
* Analyze regional variations
* Apply machine learning for deeper insights
